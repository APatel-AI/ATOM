What's Atom

ATOM is a Three.js and Gsap-based project which is meant to be an interactive yet simple web animation. Inspired by courses conducted on the Youtube channel developedbyed , I have created my own version of an Introductory Three.js project which attempts to mimic my concept of an ATOM. This project will consistently be altered as I work on more projects and learn other concepts.
